## 소개
일본에서 기타나 베이스 등 기재를 구매 할 때 관세, 부가세, 택스 프리 등  
번거로운 계산을 간편히 하기 위해 가격(엔)만 입력하면 필요한 금액을 계산해주는 웹 사이트 입니다.

## 기능  
- 가격 입력 시 환율 계산
    - 관세
    - 부가세
    - 관세 감면
    - 택스 프리/리펀
- 디지마크 링크 입력시 가격 자동 입력
